Сервис сокращения ссылок

### Цель данного проекта - тренировка в следующих технологиях:

Backend: Java 17+, Spring Boot 3.x

База данных: PostgreSQL (+ Flyway для миграций)

Кэш: Redis (для быстрого редиректа)

Frontend: Thymeleaf + Bootstrap 5 (минималистичный UI)

Сборка: Maven/Gradle

